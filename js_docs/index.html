# JS简介

JavaScript是一种动态弱类型语言.作为开发Web页面的脚本语言而出名,但目前已经被用到了很多非浏览器环境.

JavaScript的标准是ECMAScript.目前使用较为广泛的是ES6,也叫ES2015.并且在此版本后,ECMAScript每年发布一次新标准,最新的为ECMAScript2020
