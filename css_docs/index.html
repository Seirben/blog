# css简介

css的全称是层叠样式表(Cascading Style Sheets),用来描述HTML等文档的呈现.

由W3C规范实现跨浏览器的标准化.

目前有多种css预编译语言,例如scss,less,stylus 等可以帮助我们写出更快更优雅的样式语言.
